# Orbiting Circles

A description for Orbiting Circles component.

## Installation

```bash
npx shadcn-svelte@latest add https://sv-animations.vercel.app/r/orbiting-circles.json
```

## Usage

```svelte
<script lang="ts">
  import OrbitingCircles from "$lib/components/magic-ui/orbiting-circles/orbiting-circles.svelte";
</script>

<OrbitingCircles>
  <!-- Your content here -->
</OrbitingCircles>
```

## Props

| Prop    | Type     | Default | Description                     |
| ------- | -------- | ------- | ------------------------------- |
| `class` | `string` | `""`    | Additional CSS classes to apply |
