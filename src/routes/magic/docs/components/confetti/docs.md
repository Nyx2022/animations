# Confetti

A description for Confetti component.

## Installation

```bash
npx shadcn-svelte@latest add https://animations.sikandarjodd.dev/r/confetti.json
```

## Usage

```svelte
<script lang="ts">
  import Confetti from "$lib/components/magic-ui/confetti/confetti.svelte";
</script>

<Confetti>
  <!-- Your content here -->
</Confetti>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `class` | `string` | `""` | Additional CSS classes to apply |
