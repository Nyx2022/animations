# Progressive Blur

A description for Progressive Blur component.

## Installation

```bash
npx shadcn-svelte@latest add https://sv-animations.vercel.app/r/progressive-blur.json
```

## Usage

```svelte
<script lang="ts">
  import ProgressiveBlur from "$lib/components/magic-ui/progressive-blur/progressive-blur.svelte";
</script>

<ProgressiveBlur>
  <!-- Your content here -->
</ProgressiveBlur>
```

## Props

| Prop    | Type     | Default | Description                     |
| ------- | -------- | ------- | ------------------------------- |
| `class` | `string` | `""`    | Additional CSS classes to apply |
