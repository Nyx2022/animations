# Terminal

A description for Terminal component.

## Installation

```bash
npx shadcn-svelte@latest add https://animations.sikandarjodd.dev/r/terminal.json
```

## Usage

```svelte
<script lang="ts">
  import Terminal from "$lib/components/magic-ui/terminal/terminal.svelte";
</script>

<Terminal>
  <!-- Your content here -->
</Terminal>
```

## Props

| Prop    | Type     | Default | Description                     |
| ------- | -------- | ------- | ------------------------------- |
| `class` | `string` | `""`    | Additional CSS classes to apply |
