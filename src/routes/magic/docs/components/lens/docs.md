# Lens

A description for Lens component.

## Installation

```bash
npx shadcn-svelte@latest add https://animations.sikandarjodd.dev/r/lens.json
```

## Usage

```svelte
<script lang="ts">
  import Lens from "$lib/components/magic-ui/lens/lens.svelte";
</script>

<Lens>
  <!-- Your content here -->
</Lens>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `class` | `string` | `""` | Additional CSS classes to apply |
