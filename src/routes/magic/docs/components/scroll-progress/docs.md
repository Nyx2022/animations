# Scroll Progress

A description for Scroll Progress component.

## Installation

```bash
npx shadcn-svelte@latest add https://animations.sikandarjodd.dev/r/scroll-progress.json
```

## Usage

```svelte
<script lang="ts">
  import ScrollProgress from "$lib/components/magic-ui/scroll-progress/scroll-progress.svelte";
</script>

<ScrollProgress>
  <!-- Your content here -->
</ScrollProgress>
```

## Props

| Prop    | Type     | Default | Description                     |
| ------- | -------- | ------- | ------------------------------- |
| `class` | `string` | `""`    | Additional CSS classes to apply |
