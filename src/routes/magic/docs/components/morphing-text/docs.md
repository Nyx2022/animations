# Morphing Text

A description for Morphing Text component.

## Installation

```bash
npx shadcn-svelte@latest add https://animations.sikandarjodd.dev/r/morphing-text.json
```

## Usage

```svelte
<script lang="ts">
  import MorphingText from "$lib/components/magic-ui/morphing-text/morphing-text.svelte";
</script>

<MorphingText>
  <!-- Your content here -->
</MorphingText>
```

## Props

| Prop    | Type     | Default | Description                     |
| ------- | -------- | ------- | ------------------------------- |
| `class` | `string` | `""`    | Additional CSS classes to apply |
