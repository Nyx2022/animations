# Hero Video Dialog

A description for Hero Video Dialog component.

## Installation

```bash
npx shadcn-svelte@latest add https://animations.sikandarjodd.dev/r/hero-video-dialog.json
```

## Usage

```svelte
<script lang="ts">
  import HeroVideoDialog from "$lib/components/magic-ui/hero-video-dialog/hero-video-dialog.svelte";
</script>

<HeroVideoDialog>
  <!-- Your content here -->
</HeroVideoDialog>
```

## Props

| Prop    | Type     | Default | Description                     |
| ------- | -------- | ------- | ------------------------------- |
| `class` | `string` | `""`    | Additional CSS classes to apply |
