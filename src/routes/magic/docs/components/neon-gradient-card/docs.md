# Neon Gradient Card

A description for Neon Gradient Card component.

## Installation

```bash
npx shadcn-svelte@latest add https://animations.sikandarjodd.dev/r/neon-gradient-card.json
```

## Usage

```svelte
<script lang="ts">
  import NeonGradientCard from "$lib/components/magic-ui/neon-gradient-card/neon-gradient-card.svelte";
</script>

<NeonGradientCard>
  <!-- Your content here -->
</NeonGradientCard>
```

## Props

| Prop    | Type     | Default | Description                     |
| ------- | -------- | ------- | ------------------------------- |
| `class` | `string` | `""`    | Additional CSS classes to apply |
