# Warp Background

A description for Warp Background component.

## Installation

```bash
npx shadcn-svelte@latest add https://animations.sikandarjodd.dev/r/warp-background.json
```

## Usage

```svelte
<script lang="ts">
  import WarpBackground from "$lib/components/magic-ui/warp-background/warp-background.svelte";
</script>

<WarpBackground>
  <!-- Your content here -->
</WarpBackground>
```

## Props

| Prop    | Type     | Default | Description                     |
| ------- | -------- | ------- | ------------------------------- |
| `class` | `string` | `""`    | Additional CSS classes to apply |
