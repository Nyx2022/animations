# Word Rotate

A description for Word Rotate component.

## Installation

```bash
npx shadcn-svelte@latest add https://animations.sikandarjodd.dev/r/word-rotate.json
```

## Usage

```svelte
<script lang="ts">
  import WordRotate from "$lib/components/magic-ui/word-rotate/word-rotate.svelte";
</script>

<WordRotate>
  <!-- Your content here -->
</WordRotate>
```

## Props

| Prop    | Type     | Default | Description                     |
| ------- | -------- | ------- | ------------------------------- |
| `class` | `string` | `""`    | Additional CSS classes to apply |
