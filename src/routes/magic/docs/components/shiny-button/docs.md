# Shiny Button

A description for Shiny Button component.

## Installation

```bash
npx shadcn-svelte@latest add https://animations.sikandarjodd.dev/r/shiny-button.json
```

## Usage

```svelte
<script lang="ts">
  import ShinyButton from "$lib/components/magic-ui/shiny-button/shiny-button.svelte";
</script>

<ShinyButton>
  <!-- Your content here -->
</ShinyButton>
```

## Props

| Prop    | Type     | Default | Description                     |
| ------- | -------- | ------- | ------------------------------- |
| `class` | `string` | `""`    | Additional CSS classes to apply |
