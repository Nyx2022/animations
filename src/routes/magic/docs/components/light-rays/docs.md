# Light Rays

A description for Light Rays component.

## Installation

```bash
npx shadcn-svelte@latest add https://animations.sikandarjodd.dev/r/light-rays.json
```

## Usage

```svelte
<script lang="ts">
  import LightRays from "$lib/components/magic-ui/light-rays/light-rays.svelte";
</script>

<LightRays>
  <!-- Your content here -->
</LightRays>
```

## Props

| Prop    | Type     | Default | Description                     |
| ------- | -------- | ------- | ------------------------------- |
| `class` | `string` | `""`    | Additional CSS classes to apply |
