# Line Shadow Text

A description for Line Shadow Text component.

## Installation

```bash
npx shadcn-svelte@latest add https://animations.sikandarjodd.dev/r/line-shadow-text.json
```

## Usage

```svelte
<script lang="ts">
  import LineShadowText from "$lib/components/magic-ui/line-shadow-text/line-shadow-text.svelte";
</script>

<LineShadowText>
  <!-- Your content here -->
</LineShadowText>
```

## Props

| Prop    | Type     | Default | Description                     |
| ------- | -------- | ------- | ------------------------------- |
| `class` | `string` | `""`    | Additional CSS classes to apply |
