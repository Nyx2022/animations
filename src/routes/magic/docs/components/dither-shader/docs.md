# Dither Shader

A description for Dither Shader component.

## Installation

```bash
npx shadcn-svelte@latest add https://sv-animations.vercel.app/r/dither-shader.json
```

## Usage

```svelte
<script lang="ts">
  import DitherShader from "$lib/components/magic-ui/dither-shader/dither-shader.svelte";
</script>

<DitherShader>
  <!-- Your content here -->
</DitherShader>
```

## Props

| Prop    | Type     | Default | Description                     |
| ------- | -------- | ------- | ------------------------------- |
| `class` | `string` | `""`    | Additional CSS classes to apply |
