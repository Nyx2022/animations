# Magic Card

A description for Magic Card component.

## Installation

```bash
npx shadcn-svelte@latest add https://sv-animations.vercel.app/r/magic-card.json
```

## Usage

```svelte
<script lang="ts">
  import MagicCard from "$lib/components/magic-ui/magic-card/magic-card.svelte";
</script>

<MagicCard>
  <!-- Your content here -->
</MagicCard>
```

## Props

| Prop    | Type     | Default | Description                     |
| ------- | -------- | ------- | ------------------------------- |
| `class` | `string` | `""`    | Additional CSS classes to apply |
