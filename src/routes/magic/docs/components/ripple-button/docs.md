# Ripple Button

A description for Ripple Button component.

## Installation

```bash
npx shadcn-svelte@latest add https://animations.sikandarjodd.dev/r/ripple-button.json
```

## Usage

```svelte
<script lang="ts">
  import RippleButton from "$lib/components/magic-ui/ripple-button/ripple-button.svelte";
</script>

<RippleButton>
  <!-- Your content here -->
</RippleButton>
```

## Props

| Prop    | Type     | Default | Description                     |
| ------- | -------- | ------- | ------------------------------- |
| `class` | `string` | `""`    | Additional CSS classes to apply |
