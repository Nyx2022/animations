# Pulsating Button

A description for Pulsating Button component.

## Installation

```bash
npx shadcn-svelte@latest add https://animations.sikandarjodd.dev/r/pulsating-button.json
```

## Usage

```svelte
<script lang="ts">
  import PulsatingButton from "$lib/components/magic-ui/pulsating-button/pulsating-button.svelte";
</script>

<PulsatingButton>
  <!-- Your content here -->
</PulsatingButton>
```

## Props

| Prop    | Type     | Default | Description                     |
| ------- | -------- | ------- | ------------------------------- |
| `class` | `string` | `""`    | Additional CSS classes to apply |
