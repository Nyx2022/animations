# Video Text

A description for Video Text component.

## Installation

```bash
npx shadcn-svelte@latest add https://animations.sikandarjodd.dev/r/video-text.json
```

## Usage

```svelte
<script lang="ts">
  import VideoText from "$lib/components/magic-ui/video-text/video-text.svelte";
</script>

<VideoText>
  <!-- Your content here -->
</VideoText>
```

## Props

| Prop    | Type     | Default | Description                     |
| ------- | -------- | ------- | ------------------------------- |
| `class` | `string` | `""`    | Additional CSS classes to apply |
