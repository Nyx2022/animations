# Shimmer Button

A description for Shimmer Button component.

## Installation

```bash
npx shadcn-svelte@latest add https://animations.sikandarjodd.dev/r/shimmer-button.json
```

## Usage

```svelte
<script lang="ts">
  import ShimmerButton from "$lib/components/magic-ui/shimmer-button/shimmer-button.svelte";
</script>

<ShimmerButton>
  <!-- Your content here -->
</ShimmerButton>
```

## Props

| Prop    | Type     | Default | Description                     |
| ------- | -------- | ------- | ------------------------------- |
| `class` | `string` | `""`    | Additional CSS classes to apply |
