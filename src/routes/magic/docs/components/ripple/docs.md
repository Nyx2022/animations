# Ripple

A description for Ripple component.

## Installation

```bash
npx shadcn-svelte@latest add https://animations.sikandarjodd.dev/r/ripple.json
```

## Usage

```svelte
<script lang="ts">
  import Ripple from "$lib/components/magic-ui/ripple/ripple.svelte";
</script>

<Ripple>
  <!-- Your content here -->
</Ripple>
```

## Props

| Prop    | Type     | Default | Description                     |
| ------- | -------- | ------- | ------------------------------- |
| `class` | `string` | `""`    | Additional CSS classes to apply |
