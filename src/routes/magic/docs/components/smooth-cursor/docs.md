# Smooth Cursor

A description for Smooth Cursor component.

## Installation

```bash
npx shadcn-svelte@latest add https://animations.sikandarjodd.dev/r/smooth-cursor.json
```

## Usage

```svelte
<script lang="ts">
  import SmoothCursor from "$lib/components/magic-ui/smooth-cursor/smooth-cursor.svelte";
</script>

<SmoothCursor>
  <!-- Your content here -->
</SmoothCursor>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `class` | `string` | `""` | Additional CSS classes to apply |
