# Rainbow Button

A description for Rainbow Button component.

## Installation

```bash
npx shadcn-svelte@latest add https://animations.sikandarjodd.dev/r/rainbow-button.json
```

## Usage

```svelte
<script lang="ts">
  import RainbowButton from "$lib/components/magic-ui/rainbow-button/rainbow-button.svelte";
</script>

<RainbowButton>
  <!-- Your content here -->
</RainbowButton>
```

## Props

| Prop    | Type     | Default | Description                     |
| ------- | -------- | ------- | ------------------------------- |
| `class` | `string` | `""`    | Additional CSS classes to apply |
