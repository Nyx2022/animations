# Hyper Text

A description for Hyper Text component.

## Installation

```bash
npx shadcn-svelte@latest add https://animations.sikandarjodd.dev/r/hyper-text.json
```

## Usage

```svelte
<script lang="ts">
  import HyperText from "$lib/components/magic-ui/hyper-text/hyper-text.svelte";
</script>

<HyperText>
  <!-- Your content here -->
</HyperText>
```

## Props

| Prop    | Type     | Default | Description                     |
| ------- | -------- | ------- | ------------------------------- |
| `class` | `string` | `""`    | Additional CSS classes to apply |
