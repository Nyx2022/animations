# Dock

A description for Dock component.

## Installation

```bash
npx shadcn-svelte@latest add https://sv-animations.vercel.app/r/dock.json
```

## Usage

```svelte
<script lang="ts">
  import Dock from "$lib/components/magic-ui/dock/dock.svelte";
</script>

<Dock>
  <!-- Your content here -->
</Dock>
```

## Props

| Prop    | Type     | Default | Description                     |
| ------- | -------- | ------- | ------------------------------- |
| `class` | `string` | `""`    | Additional CSS classes to apply |
