# Bento Grid

A description for Bento Grid component.

## Installation

```bash
npx shadcn-svelte@latest add https://animations.sikandarjodd.dev/r/bento-grid.json
```

## Usage

```svelte
<script lang="ts">
  import BentoGrid from "$lib/components/magic-ui/bento-grid/bento-grid.svelte";
</script>

<BentoGrid>
  <!-- Your content here -->
</BentoGrid>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `class` | `string` | `""` | Additional CSS classes to apply |
