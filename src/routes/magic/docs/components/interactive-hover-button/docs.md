# Interactive Hover Button

A description for Interactive Hover Button component.

## Installation

```bash
npx shadcn-svelte@latest add https://animations.sikandarjodd.dev/r/interactive-hover-button.json
```

## Usage

```svelte
<script lang="ts">
  import InteractiveHoverButton from "$lib/components/magic-ui/interactive-hover-button/interactive-hover-button.svelte";
</script>

<InteractiveHoverButton>
  <!-- Your content here -->
</InteractiveHoverButton>
```

## Props

| Prop    | Type     | Default | Description                     |
| ------- | -------- | ------- | ------------------------------- |
| `class` | `string` | `""`    | Additional CSS classes to apply |
