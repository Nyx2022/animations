# Icon Cloud

A description for Icon Cloud component.

## Installation

```bash
npx shadcn-svelte@latest add https://sv-animations.vercel.app/r/icon-cloud.json
```

## Usage

```svelte
<script lang="ts">
  import IconCloud from "$lib/components/magic-ui/icon-cloud/icon-cloud.svelte";
</script>

<IconCloud>
  <!-- Your content here -->
</IconCloud>
```

## Props

| Prop    | Type     | Default | Description                     |
| ------- | -------- | ------- | ------------------------------- |
| `class` | `string` | `""`    | Additional CSS classes to apply |
