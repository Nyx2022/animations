# Number Ticker

A description for Number Ticker component.

## Installation

```bash
npx shadcn-svelte@latest add https://animations.sikandarjodd.dev/r/number-ticker.json
```

## Usage

```svelte
<script lang="ts">
  import NumberTicker from "$lib/components/magic-ui/number-ticker/number-ticker.svelte";
</script>

<NumberTicker>
  <!-- Your content here -->
</NumberTicker>
```

## Props

| Prop    | Type     | Default | Description                     |
| ------- | -------- | ------- | ------------------------------- |
| `class` | `string` | `""`    | Additional CSS classes to apply |
